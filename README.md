# Real-Time Tracking App

## Overview

This repository contains a **Real-Time Tracking Web Application** built with **Node.js**, **Express.js**, **Socket.IO**, and **EJS**.  
The application allows users to share and view real-time location updates on an interactive map.

Users can connect from their browser, share their current geolocation, and view the live positions of other connected users.

## Features

- **Real-Time Location Updates** – Clients broadcast their current coordinates to the server and all connected users.  
- **Interactive Map View** – Visualize user locations on a map in real time.  
- **Socket Communication** – Uses WebSockets via **Socket.IO** for bidirectional, event-based communication.  
- **Responsive UI** – Works on desktop and mobile browsers.

# How to Run 

1. Clone the Repo on your local system.

2. Open terminal and run this command

       npm intall
       npx nodemon app.js

3. Go to any web browser and type

       localhost:3000

4. Done, your code is running on port 3000.
